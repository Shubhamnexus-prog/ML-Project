# Student Marks Predictor

A Machine Learning web application that predicts student marks based on input features. This project demonstrates the complete machine learning workflow, including data preprocessing, model training, model serialization, and deployment using Flask.

## Features

* Predict student marks using a trained machine learning model
* User-friendly web interface
* Flask-based backend
* Real-time predictions
* Easy deployment and scalability

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Flask
* HTML
* CSS


## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Shubhamnexus-prog/ML-Project.git
   ```

2. Navigate to the project directory:

   ```bash
   cd ML-Project
   ```

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the application:

   ```bash
   python app.py
   ```

5. Open your browser and visit:

   ```text
   http://127.0.0.1:5000
   ```
